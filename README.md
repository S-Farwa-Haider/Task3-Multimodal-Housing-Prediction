# Task 3: Multimodal Housing Price Prediction

## Objective

The objective of this project is to predict house prices using both structured property data and house images, combining information from multiple data modalities.

## Methodology / Approach

* Loaded housing data containing property features such as area, bedrooms, and bathrooms.
* Loaded house images and performed preprocessing.
* Used a Convolutional Neural Network (CNN) to extract image features.
* Processed tabular features separately.
* Combined image and tabular features using feature fusion techniques.
* Trained a regression model to predict house prices.
* Evaluated model performance using MAE and RMSE.

## Key Results / Observations

* Successfully combined image and structured data into a single predictive model.
* Demonstrated the advantages of multimodal machine learning.
* Learned how CNNs can extract meaningful visual features.
* Evaluated prediction accuracy using regression metrics.
